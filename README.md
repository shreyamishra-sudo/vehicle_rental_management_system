# Vehicle Rental Management System

A full-stack vehicle rental management system built to understand real-world application flow across frontend, backend, and database layers.

The project focuses on practical implementation of user workflows such as vehicle search, booking, and return processing.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MySQL

## Key Features
- Vehicle availability search based on date range
- Customer registration and login
- Rental booking and return handling
- Backend APIs with database integration

## Project Scope
This project was built as a learning-focused full-stack application.
It emphasizes clean structure, API design, and database interaction rather than production deployment.

## Project Structure
- `frontend/` – React-based user interface
- `backend/` – Express.js server and APIs
- `vehicle_rental_mysql.sql` – Database schema
